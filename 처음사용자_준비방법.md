# GHJ Codex V.03 처음 사용자 준비 방법

이 프로그램은 실행 후 웹 화면에서 KRX 아이디, KRX 비밀번호, OpenAPI 키를 입력해야 데이터 수집이 시작됩니다.

## 1. KRX 아이디/비밀번호 준비

1. KRX 정보데이터시스템에 접속합니다.
   - https://data.krx.co.kr/
2. 회원가입을 진행합니다.
3. 가입한 KRX 아이디와 비밀번호를 준비합니다.
4. 프로그램 웹 화면의 `KRX 아이디`, `KRX 비밀번호` 칸에 입력합니다.

## 2. OpenAPI 키 발급

1. KRX OpenAPI 전용 사이트에 접속합니다.
   - https://openapi.krx.co.kr/
2. KRX 계정으로 로그인합니다.
3. OpenAPI 사이트 안에서 `인증키`, `API Key`, `활용신청`, `OpenAPI 신청` 관련 메뉴를 찾습니다.
4. OpenAPI 이용 신청 또는 인증키 발급 절차를 진행합니다.
5. 발급된 키를 복사합니다.
6. 프로그램 웹 화면의 `OpenAPI 키` 칸에 붙여넣습니다.

KRX 정보데이터시스템 메인에서도 OpenAPI 링크로 이동할 수 있습니다.

- https://data.krx.co.kr/

## 3. 프로그램 실행

PowerShell에서 아래 순서대로 실행합니다.

```powershell
cd "C:\Users\MK-Desktop\Documents\외국인,기관수급현황실시간대시보드 제작하기"
python ghj_codex_V_03.py
```

브라우저가 자동으로 열리며, 열리지 않으면 아래 주소로 접속합니다.

```text
http://127.0.0.1:5000
```

## 4. 결과 파일 위치

수집이 완료되면 웹 화면에서 `엑셀 다운로드` 버튼을 누를 수 있습니다.

파일은 아래 폴더에도 자동 저장됩니다.

```text
outputs\YYYYMMDD\YYYY-MM-DD_통합파일.xlsx
```

## 참고

KRX 사이트 메뉴명과 발급 절차는 사이트 개편에 따라 달라질 수 있습니다. 메뉴가 보이지 않으면 KRX OpenAPI 사이트 안에서 `OpenAPI`, `API Key`, `인증키`, `활용신청` 단어로 검색해 확인하세요.
